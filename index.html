Proxtor - Outil de Sécurisation de Kali Linux

Bienvenue sur la page de présentation de l'outil Proxtor ! Cet outil permet de sécuriser votre Kali Linux en configurant différents services comme le DNS, le VPN, le proxy, et Tor.

Fonctionnalités de l'outil :

  • Changer les DNS : Sélectionnez un serveur DNS rapide et sécurisé comme Google, Cloudflare, ou un DNS personnalisé.
  • Configurer un VPN : Connectez-vous à un VPN pour sécuriser votre connexion Internet.
  • Configurer un Proxy : Utilisez un proxy pour masquer votre adresse IP.
  • Configurer Tor : Utilisez Tor pour anonymiser vos connexions en ligne.

Comment utiliser l'outil :

Pour utiliser cet outil, vous pouvez le télécharger directement en une commande avec l'une des méthodes suivantes :

        curl -s https://gist.githubusercontent.com/trh4ckn0n/a8e71e57d1e7ee42013f1713ac1a5ac8/raw/dc19399c6875d871bc6f395aafa3670f4dca86de/proxtor.py | python3

Ou bien, vous pouvez utiliser wget de la manière suivante :

        wget -qO- https://gist.githubusercontent.com/trh4ckn0n/a8e71e57d1e7ee42013f1713ac1a5ac8/raw/dc19399c6875d871bc6f395aafa3670f4dca86de/proxtor.py | python3
